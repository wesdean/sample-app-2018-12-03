Note Pad

This is a simple note taking app loaded with some demo notes. Add, edit, and delete notes. When you are done you can export the notes to JSON and do with them what you will. To import saved notes, paste the JSON into the import box then click the "Import Notes" button. Viola! Your notes are loaded.
